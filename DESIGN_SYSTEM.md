# Optiphoto

## What is Optiphoto?

Optiphoto is a tool that allows you to optimize your photos for the web. It is a simple and easy to use tool that allows you to optimize your photos for the web, by perfoming various transformations, and retrieving the images in different formats. Also it includes user authentication and authorization.

## Requirements and goasl of the system

**Funcional Requirements:**

- User authentication
- Users should be able to upload/download/view their photos
- Image management
- Image transformation

**Non-Functional Requirements:**

- Performance
- Security
- Scalability
- Reliability
- Availability

## Capacity storage estimation and constraints

The capacity storage is estimated to be 100GB, with a maximum of 1000 photos per user.
